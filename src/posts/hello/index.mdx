# Deploy Gatsby

---

<img className="logo" src={Logo} alt="로고" />

**Netlify** - GitHub에 소스 파일을 저장하고 이를 빌드하여 결과물을 CDN 통해 웹사이트 호스팅 업체

## 선정 이유

1. DNS 적용 가능

Github.io와 다르게 metaverse.bluedigm.com 사용자 정의 도메인 사용

2. 지속적인 배포

지속적인 배포를 활용하여 사이트 자산을 생성하는 자동화된 빌드 프로세스 시작

프로젝트를 깃허브 원격 저장소에 업로드하면 자동 배포 수행

3. 무료 사용 가능

정해진 리소스가 있지만 임계점을 넘을 일은 없을거라 판단

한달에 빌드 시간이 300분 이상, 월 트래픽 100G 이상이 되어야 과금 발생
